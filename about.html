Task

  • Based on available data on a chemical compound, predict the value of its Selectivity Index (SI) or IC50/CC50.
  • Predicting only the IC50 or CC50 value is acceptable.
  • It is possible to solve the classification problem into a significant number of classes rather than predicting the exact values of the specified indices.

Result Format

  • .ipynb file with an overview of the provided (or additionally found) data
  • .ipynb file with the model(s) and training procedure, comparison of different models, and identification of the best ones.
  • Detailed description of the data preprocessing procedure considering various options
  • Additional program files in a convenient format

Data

  • Two datasets are provided - ~1400 compounds with calculated CC50/IC50/SI against the same H1N1 strain and ~35000 compounds with calculated IC50 against different H1N1 strains.
  • The primary focus should be on working with the provided data.

Further Use

  • In case of significant results on a large volume of available and found data, the solution materials can be used for the research task of building machine learning models to predict the selectivity index.
